# Python Registration & Login System
A simple console-based user authentication system built using Python.  
This project allows users to register,login, and recover forgotten passwords using file handling instead of a database.

# Project Overview
The Registration & Login System simulates how real-world authentication works.  
User credentials are validated and stored securely in a text file, ensuring data persistence even after the program exits.
This project is ideal for to understand authentication logic, validation, and file handling in Python.

# Features
-  User Registration with email validation  
- Strong Password Validation  
- Secure storage using file handling  
- Login authentication  
- Forgot Password functionality  
- Simple and user-friendly menu  

# Technologies Used
- Python
- Regular Expressions (re) – for email & password validation
- File Handling – to store user credentials
- OS Module – to check file existence
